# Security Policy

## Supported Versions

This project is provided as a static client-side tool. Security updates are applied to the latest version available in this repository.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Please open a **private security report** via GitHub or contact the maintainer directly.

Do not publicly disclose vulnerabilities before they have been reviewed and addressed.

## Scope

This tool runs entirely in the browser and does not transmit or store any user data. However, potential issues such as client-side script injection or malicious file handling should still be reported.

Thank you for helping keep this project secure.
